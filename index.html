Gemini API キーの設定

レシピを提案するには Google Gemini の API キーが必要です。

1クリックでAPIキーを取得（無料）

簡単3ステップ取得方法：

  1. 上の緑色のボタンから Google AI Studio に進みます。
  2. 「Create API key」を押して生成されたキー（AIzaSy...）をコピーします。
  3. コピーしたキーを下の欄に貼り付けて「設定を保存する」を押します。
入力されたキーはブラウザ（ローカルストレージ）にのみ保存され、外部に送信されることはありません。
